# movel-poc-web

Projeto Proof of Concept (PoC) de frontend web para o sistema "movel".  
Este repositório contém a base da aplicação web, exemplos de componentes e instruções de setup.

## Descrição
Aplicação web POC com foco em demonstrar funcionalidades básicas, layout e integração com APIs. Ajuste este README conforme o projeto evoluir.

## Pré-requisitos
- Node.js 18+ e npm ou yarn
- Git

## Instalação e execução (local)
1. Clone o repositório:
   git clone https://github.com/TiagoAlves14/movel-poc-web.git
2. Entre na pasta do projeto:
   cd movel-poc-web
3. Crie a branch main localmente e adicione o README (se ainda não existir):
   git checkout -b main
   echo "# movel-poc-web" > README.md
   git add README.md
   git commit -m "chore: initial commit"
4. Configure o remote (se ainda não estiver configurado) e envie para o GitHub:
   git remote add origin https://github.com/TiagoAlves14/movel-poc-web.git
   git push -u origin main

Se já houver arquivos locais que você queira subir, faça git add/commit normalmente antes do git push.
